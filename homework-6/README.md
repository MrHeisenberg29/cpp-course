# Поиск циклов длины 4 в бинарной матрице

## 1. Общая информация

Дана бинарная матрица `H` размера `N × M`, где каждый элемент `H[i][j]` равен `0` или `1`.

Рассмотрим двудольный граф:

- левая доля — строки матрицы `V = {1, 2, ..., N}`,
- правая доля — столбцы матрицы `C = {1, 2, ..., M}`,
- ребро между вершинами `i ∈ V` и `j ∈ C` существует тогда и только тогда, когда `H[i][j] = 1`.

**Циклом длины 4** в таком графе называется последовательность вершин  
`v1 → c1 → v2 → c2 → v1`,  
где:

- `v1` и `v2` — различные строки,
- `c1` и `c2` — различные столбцы,
- `H[v1][c1] = 1`,
- `H[v1][c2] = 1`,
- `H[v2][c1] = 1`,
- `H[v2][c2] = 1`.

Эквивалентно, в матрице существует подматрица размера `2 × 2`, у которой все четыре угловых элемента равны `1`.

---

## 2. Требования

По заданной бинарной матрице определить, **существует ли хотя бы один цикл длины 4**.
Программа должна принимать входные данные в виде пути к файлу первым аргументом командной строки или из стандартного входного потока, если этот аргумент не передан.

---

### 2.1 Формат входных данных

- В первой строке заданы два целых числа `N` и `M`  
  (`1 ≤ N, M ≤ 2⋅10^5` — верхние границы приведены ориентировочно и зависят от ограничений по памяти и времени).

- Далее следуют `N` строк, каждая из которых содержит ровно `M` символов `0` или `1` без пробелов — строки матрицы.

---

### 2.2 Формат выходных данных

Выведите:

- `1`, если в матрице существует цикл длины 4;
- `0`, если ни одного такого цикла нет.

---

## 3. Пояснение

Цикл длины 4 существует тогда и только тогда, когда найдутся две различные строки, имеющие **как минимум два общих столбца**, в которых стоят единицы.

Эквивалентная формулировка:  
существует пара различных столбцов `(c1, c2)`, которая встречается (оба равны `1`) **как минимум в двух различных строках**.

---

## 4. Пример

### Вход
```
4 5
00000
00110
00110
00000
```

### Выход
```
1
```
